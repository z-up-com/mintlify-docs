## [​](#認証)認証

[​](#param-authorization)Authorizationstring必須すべてのAPIエンドポイントでBearer Tokenによる認証が必要ですAPI Keyの取得：[API Key管理ページ](https://dashlyai.cc/console/token)にアクセスしてAPI Keyを取得してください使用時はリクエストヘッダーに追加：コピー
```shiki
Authorization: Bearer YOUR_API_KEY

```
## [​](#リクエストボディ)リクエストボディ

[​](#param-model)modelstringデフォルト:"veo3.1-fast"必須動画生成モデル名利用可能なモデル：

- `veo3.1-fast` - 高速生成モデル、クイックプレビューと反復処理に適しています
- `veo3.1-quality` - 高品質生成モデル、最終制作に適しています

Example: `"veo3.1-fast"`
[​](#param-prompt)promptstring必須動画生成のテキスト説明
[​](#param-duration)durationinteger動画の長さ（秒）固定値：`8`（VEO3 は 8 秒の長さのみサポート）
[​](#param-aspect-ratio)aspect\_ratiostring動画のアスペクト比サポートされている形式：

- `16:9` (横向き)
- `9:16` (縦向き)

[​](#param-image-urls)image\_urlsarray動画生成に使用する参考画像URLの配列**💡 クイック入力（Try itエリア）：**

1. ”+ Add an item” をクリックして画像URLを追加
2. 画像の完全なURLアドレスを入力

表示 フィールド詳細

[​](#param-url)urlstring必須画像URLアドレス以下が可能：

- 公開アクセス可能な画像URL（http:// または https://）
- Base64エンコードされた画像データ

**制限：**

- 最大ファイルサイズ：10MB
- サポート形式：.jpeg、.jpg、.png、.webp
