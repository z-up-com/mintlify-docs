  --header 'Authorization: Bearer <token>'

## [​](#認証)認証

[​](#param-authorization)Authorizationstring必須すべてのAPIエンドポイントにはBearer Token認証が必要ですAPIキーの取得:[APIキー管理ページ](https://dashlyai.cc/console/token)にアクセスしてAPIキーを取得してくださいリクエストヘッダーに追加:コピー
```shiki
Authorization: Bearer YOUR_API_KEY

```
## [​](#パスパラメータ)パスパラメータ

[​](#param-task-id)task\_idstring必須生成APIから返されたタスクID
## [​](#クエリパラメータ)クエリパラメータ

[​](#param-language)languagestringレスポンスコンテンツの言語、以下の値をサポート：

- `zh` - 中国語
- `en` - 英語
- `ko` - 韓国語
- `ja` - 日本語

デフォルトは英語
## [​](#レスポンス)レスポンス

[​](#param-id)idstring一意のタスク識別子
[​](#param-status)statusstringタスクステータスの値:

- `pending` - 処理待ち
- `processing` - 処理中
- `completed` - 正常に完了
- `failed` - 失敗
- `cancelled` - ユーザーによってキャンセル

[​](#param-progress)progressintegerタスク進捗率 (0–100)
[​](#param-result)resultobjectタスクの結果、ステータスが`completed`の場合のみ返されます

表示 プロパティ

[​](#param-images)imagesarray生成された画像オブジェクトの配列（画像生成タスクの場合）[​](#param-videos)videosarray生成された動画オブジェクトの配列（動画生成タスクの場合）[​](#param-thumbnail-url)thumbnail\_urlstring動画のサムネイルURL（動画生成タスクの場合）**重要：** このフィールドは、タスク作成時にリクエストパラメータでサムネイル生成を明示的に指定した場合のみ返されます。サムネイル生成がリクエストされていない場合、このフィールドはレスポンスに含まれません。
[​](#param-created)createdintegerタスク作成タイムスタンプ
[​](#param-completed)completedintegerタスク完了タイムスタンプ（完了時のみ存在）
[​](#param-estimated-time)estimated\_timeinteger推定完了時間（秒）
[​](#param-actual-time)actual\_timeinteger実際の完了時間（秒）（完了時のみ存在）
[​](#param-error)errorobjectエラー詳細（ステータスが`failed`の場合のみ存在）

表示 プロパティ
