## [​](#認証)認証

[​](#param-authorization)Authorizationstring必須すべてのAPIエンドポイントはBearer Token認証が必要ですAPIキーの取得方法：[APIキー管理ページ](https://dashlyai.cc/console/token)にアクセスしてAPIキーを取得してくださいリクエストヘッダーに追加：コピー
```shiki
Authorization: Bearer YOUR_API_KEY

```
## [​](#リクエストボディ)リクエストボディ

[​](#param-model)modelstringデフォルト:"gpt-4o-image"必須画像生成モデル名Example: `"gpt-4o-image"`
[​](#param-prompt)promptstring必須画像生成のためのテキスト説明最大1000文字
[​](#param-size)sizestring画像生成サイズサポートされている形式：

- 比率：`1:1`, `2:3`, `3:2`

[​](#param-n)ninteger生成する画像の数1、2、4をサポート。数量に基づいて料金が事前に差し引かれますデフォルト：1**⚠️ 注意：** 純粋な数字を入力してください（例：`1`）。引用符を使用するとエラーになります
[​](#param-image-urls)image\_urlsarray画像から画像または画像編集のための参照画像URLリスト**💡 クイック入力（Try itエリア）：**

1. ”+ Add an item” をクリックして画像URLを追加
2. 画像の完全なURL アドレスを入力

表示 フィールド詳細

[​](#param-url)urlstring必須画像URLアドレス以下が可能：

- 公開アクセス可能な画像URL（http:// または https://）
- Base64エンコードされた画像データ

**制限：**

- 単一画像は最大 10MB
- 対応形式：.jpeg、.jpg、.png、.webp
**制限：** 最大 5 枚の画像
[​](#param-mask-url)mask\_urlstringマスク画像URL

- PNG形式である必要があります
- サイズは参照画像と一致する必要があります
- 最大4MB
