## [​](#認証)認証

[​](#param-authorization)Authorizationstring必須すべてのAPIエンドポイントはBearer Token認証が必要ですAPIキーの取得方法：[APIキー管理ページ](https://dashlyai.cc/console/token)にアクセスしてAPIキーを取得してくださいリクエストヘッダーに追加：コピー

```shiki
Authorization: Bearer YOUR_API_KEY

```
## [​](#リクエストボディ)リクエストボディ

[​](#param-model)modelstring必須モデル名サポートされているモデル：

- **OpenAI**: `gpt-5`, `gpt-5-chat-latest`, `gpt-5-mini`, `gpt-5-nano`, `gpt-5-pro`
- **Anthropic**: `claude-sonnet-4-5-20250929`, `claude-opus-4-1-20250805`, `claude-haiku-4-5-20251001`, `claude-opus-4-1-20250805-thinking`, `claude-sonnet-4-5-20250929-thinking`
- **Google**: `gemini-2.5-pro`, `gemini-2.5-flash`, `gemini-2.5-pro-thinking`, `gemini-2.5-flash-lite`
- **DeepSeek**: `deepseek-v3.1-250821`, `deepseek-v3.1-think-250821`, `deepseek-v3-0324`
- **Doubao**: `doubao-seed-1-6-251015`, `doubao-seed-1-6-flash-250828`, `doubao-seed-1-6-thinking-250715`
- 随時新しいモデルを追加中…

[​](#param-messages)messagesarray必須会話メッセージのリスト

表示 メッセージオブジェクトの構造

[​](#param-role)roleenum (string)デフォルト:"user"必須役割タイプ

- `user` - ユーザーメッセージ
- `assistant` - AI応答（複数ターン用）
- `system` - システムプロンプト
[​](#param-content)contentstring必須メッセージ内容質問や話したい内容**例：**コピー
```shiki
[{"role": "user", "content": "こんにちは、自己紹介をお願いします"}]

```
**応用的な使い方：**システムプロンプトを追加（AIに特定の役割を持たせる）：コピー
```shiki
[
  {"role": "system", "content": "あなたはプロのPython講師です"},
  {"role": "user", "content": "プログラミングの学び方を教えてください"}
]

```
複数ターンの会話（コンテキスト付き）：コピー
```shiki
[
  {"role": "user", "content": "こんにちは"},
  {"role": "assistant", "content": "こんにちは！何かお手伝いできますか？"},
  {"role": "user", "content": "AIについて教えてください"}
]

```
**役割の説明：**

- `user`: ユーザーメッセージ（ほとんどの場合これを使用）
- `system`: システムプロンプト、AIの動作や役割を設定
- `assistant`: AIの過去の返答、会話のコンテキストとして使用

[​](#param-temperature)temperaturenumber出力のランダム性を制御、範囲は0～2

- 低い値（例：0.2）は出力をより決定的にします
- 高い値（例：1.8）は出力をよりランダムにします

デフォルト：1.0
[​](#param-max-tokens)max\_tokensinteger生成する最大トークン数モデルによって最大制限が異なります。詳細は各モデルのドキュメントを参照してください
[​](#param-stream)streambooleanストリーミング出力を使用するかどうか

- `true`: ストリーミングレスポンス（SSE形式）
- `false`: 一度に完全なレスポンス

デフォルト：false
[​](#param-top-p)top\_pnumberニュークレアスサンプリングパラメータ、範囲は0～1生成されるテキストの多様性を制御します。temperatureとの併用は推奨されませんデフォルト：1.0
[​](#param-frequency-penalty)frequency\_penaltynumber頻度ペナルティ、範囲は-2.0～2.0正の値は同じ単語を繰り返す可能性を減らしますデフォルト：0
[​](#param-presence-penalty)presence\_penaltynumber存在ペナルティ、範囲は-2.0～2.0正の値は新しいトピックについて話す可能性を増やしますデフォルト：0
[​](#param-stop)stopstring or array停止シーケンス最大4つのシーケンスを指定でき、遭遇すると生成が停止します
[​](#param-n)ninteger生成する完了の数デフォルト：1**⚠️ 注意：** 純粋な数字を入力してください（例：`1`）。引用符を使用するとエラーになります
## [​](#レスポンス)レスポンス

[​](#param-id)idstringレスポンスの一意の識別子
[​](#param-object)objectstringオブジェクトタイプ、`chat.completion`に固定
[​](#param-created)createdinteger作成タイムスタンプ
[​](#param-model-1)modelstring実際に使用されたモデル名
[​](#param-choices)choicesarray生成されたレスポンスのリスト

表示 プロパティ

[​](#param-index)indexinteger選択肢のインデックス[​](#param-message)messageobjectメッセージ内容

表示 プロパティ

[​](#param-role-1)rolestringロールタイプ（assistant）[​](#param-content-1)contentstring生成されたテキスト内容[​](#param-finish-reason)finish\_reasonstring完了理由可能な値：

- `stop` - 自然な完了
- `length` - 最大長に達した
- `content_filter` - コンテンツがフィルタされた
- `function_call` - 関数呼び出し

[​](#param-usage)usageobjectトークン使用統計

表示 プロパティ

[​](#param-prompt-tokens)prompt\_tokensinteger入力メッセージのトークン数[​](#param-completion-tokens)completion\_tokensinteger生成されたコンテンツのトークン数[​](#param-total-tokens)total\_tokensinteger合計トークン数
