## [​](#認証)認証

[​](#param-authorization)Authorizationstring必須すべてのAPIエンドポイントにはBearer Token認証が必要ですAPIキーの取得:[APIキー管理ページ](https://dashlyai.cc/console/token)にアクセスしてAPIキーを取得してくださいリクエストヘッダーに追加:コピー
```shiki
Authorization: Bearer YOUR_API_KEY

```
## [​](#パスパラメータ)パスパラメータ

[​](#param-model)modelstring必須モデル名例では `gemini-2.5-pro` を使用していますが、他のサポートされている Gemini モデルに置き換えることができます：

- `gemini-2.5-flash` - Gemini 2.5 高速版
- `gemini-2.5-pro` - Gemini 2.5 専門版
- `gemini-2.5-flash-lite` - Gemini 2.5 超軽量版
- `gemini-2.5-pro-thinking` - Gemini 2.5 Pro 深層思考版

[​](#param-method)methodenum (string)必須生成方法（クイックスタートには `generateContent` を推奨）:

- `generateContent`: 完全なレスポンスを待って一度に返す
- `streamGenerateContent`: ストリーム形式で段階的にコンテンツを返す

利用可能なオプション: `generateContent`, `streamGenerateContent`
## [​](#ボディ)ボディ

[​](#param-contents)contentsarray必須会話内容のリスト最低1つのメッセージが必要

表示 contents オブジェクト構造

[​](#param-role)rolestring必須ロールタイプ:

- `user`: ユーザーメッセージ
- `model`: モデルレスポンス（会話履歴で使用）
[​](#param-parts)partsarray必須メッセージ内容の部分

表示 parts オブジェクト構造

[​](#param-text)textstringテキスト内容[​](#param-inline-data)inlineDataobjectインラインデータ（マルチモーダル入力用）

表示 inlineData プロパティ

[​](#param-mime-type)mimeTypestringMIME タイプ、例: `image/jpeg`, `image/png`[​](#param-data)datastringBase64 エンコードされたデータ例:コピー
```shiki
[
  {
```

[​](#param-generation-config)generationConfigobject生成設定（オプション）

表示 generationConfig プロパティ

[​](#param-temperature)temperaturenumber出力のランダム性を制御、範囲 0.0-2.0

- 低い値は出力をより決定的にします
- 高い値は出力をよりランダムにします

デフォルト: 1.0[​](#param-max-output-tokens)maxOutputTokensinteger生成する最大トークン数モデルによって最大制限が異なります[​](#param-top-p)topPnumberニュークリアスサンプリングパラメータ、範囲 0.0-1.0サンプリング時に考慮する確率質量を制御[​](#param-top-k)topKintegerTop-K サンプリングパラメータ各ステップで最も確率の高い K トークンからのみサンプリング[​](#param-stop-sequences)stopSequencesarray停止シーケンスのリストこれらのシーケンスに遭遇したら生成を停止
[​](#param-safety-settings)safetySettingsarray安全設定（オプション）

表示 safetySettings オブジェクト構造

[​](#param-category)categorystring安全カテゴリ:

- `HARM_CATEGORY_HATE_SPEECH`: ヘイトスピーチ
- `HARM_CATEGORY_DANGEROUS_CONTENT`: 危険なコンテンツ
- `HARM_CATEGORY_HARASSMENT`: ハラスメント
- `HARM_CATEGORY_SEXUALLY_EXPLICIT`: 性的に露骨なコンテンツ
[​](#param-threshold)thresholdstringしきい値レベル:

- `BLOCK_NONE`: ブロックしない
- `BLOCK_ONLY_HIGH`: 高リスクのみブロック
- `BLOCK_MEDIUM_AND_ABOVE`: 中リスク以上をブロック
- `BLOCK_LOW_AND_ABOVE`: 低リスク以上をブロック
