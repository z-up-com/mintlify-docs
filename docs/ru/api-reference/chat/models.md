# Поддерживаемые текстовые модели

> Все текстовые модели полностью совместимы с OpenAI Chat Completions API — просто смените параметр model

Все модели используют один и тот же API-эндпоинт. Для смены модели достаточно изменить параметр `model`:

```
POST https://toapis.com/v1/chat/completions
```

Полный список параметров запроса и формат ответа см. в [Chat Completions API](./chat).
