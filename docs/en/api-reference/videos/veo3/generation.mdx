## [​](#body)Body

[​](#param-model)modelstringdefault:"veo3.1-fast"requiredVideo generation model nameAvailable models:

- `veo3.1-fast` - Fast generation model, suitable for quick previews and iterations
- `veo3.1-quality` - High quality generation model, suitable for final production

Example: `"veo3.1-fast"`
[​](#param-prompt)promptstringrequiredText description for video generation
[​](#param-duration)durationintegerVideo duration in secondsFixed value: `8` (VEO3 only supports 8 seconds duration)
[​](#param-aspect-ratio)aspect\_ratiostringVideo aspect\_ratioSupported formats:

- `16:9` (Landscape)
- `9:16` (Portrait)

[​](#param-image-urls)image\_urlsarrayArray of reference image URLs for image-to-video generation**💡 Quick fill (Try it area):**

1. Click ”+ Add an item” to add an image URL
2. Enter the complete image URL address

Show Field details

[​](#param-url)urlstringrequiredImage URL addressCan be:

- Publicly accessible image URL (http:// or https://)
- Base64 encoded image data

**Limitations:**

- Maximum file size: 10MB
- Supported formats: .jpeg, .jpg, .png, .webp
