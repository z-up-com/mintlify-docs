## [​](#body)Body

[​](#param-model)modelstringdefault:"gpt-4o-image"requiredImage generation model nameExample: `"gpt-4o-image"`
[​](#param-prompt)promptstringrequiredText description for image generationMaximum 1000 characters
[​](#param-size)sizestringImage generation sizeSupported formats:

- Ratios: `1:1`, `2:3`, `3:2`

[​](#param-n)nintegerNumber of images to generateSupports 1, 2, 4. Charges will be pre-deducted based on the numberDefault: 1**⚠️ Note:** Must enter a plain number (e.g., `1`), do not use quotes or it will cause an error
[​](#param-image-urls)image\_urlsarrayReference image URL list for image-to-image or image editing**💡 Quick fill (Try it area):**

1. Click ”+ Add an item” to add an image URL
2. Enter the complete image URL address

Show Field details

[​](#param-url)urlstringrequiredImage URL addressCan be:

- Publicly accessible image URL (http:// or https://)
- Base64 encoded image data

**Limitations:**

- Single image up to 10MB
- Supported formats: .jpeg, .jpg, .png, .webp
**Limitation:** Maximum 5 images
[​](#param-mask-url)mask\_urlstringMask image URL

- Must be PNG format
- Size must match the reference image
- Maximum 4MB
