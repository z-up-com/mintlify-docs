## [‚Äã](#body)Body

[‚Äã](#param-model)modelstringrequiredModel nameSupported models include:

- **OpenAI**: `gpt-5`, `gpt-5-chat-latest`, `gpt-5-mini`, `gpt-5-nano`, `gpt-5-pro`
- **Anthropic**: `claude-sonnet-4-5-20250929`, `claude-opus-4-1-20250805`, `claude-haiku-4-5-20251001`, `claude-opus-4-1-20250805-thinking`, `claude-sonnet-4-5-20250929-thinking`
- **Google**: `gemini-2.5-pro`, `gemini-2.5-flash`, `gemini-2.5-pro-thinking`, `gemini-2.5-flash-lite`
- **DeepSeek**: `deepseek-v3.1-250821`, `deepseek-v3.1-think-250821`, `deepseek-v3-0324`
- **Doubao**: `doubao-seed-1-6-251015`, `doubao-seed-1-6-flash-250828`, `doubao-seed-1-6-thinking-250715`
- More models being added continuously‚Ä¶

[‚Äã](#param-messages)messagesarrayrequiredList of conversation messages

Show Message object structure

[‚Äã](#param-role)roleenum (string)default:"user"requiredRole type

- `user` - User message
- `assistant` - AI response (for multi-turn)
- `system` - System prompt
[‚Äã](#param-content)contentstringrequiredMessage contentYour question or message**Example:**Copy
```shiki
[{"role": "user", "content": "Hello, please introduce yourself"}]

```
**Advanced usage:**Add system prompt (to define AI behavior):Copy
```shiki
[
  {"role": "system", "content": "You are a professional Python tutor"},
  {"role": "user", "content": "How do I learn programming?"}
]

```
Multi-turn conversation (with context):Copy
```shiki
[
  {"role": "user", "content": "Hello"},
  {"role": "assistant", "content": "Hi! How can I help you?"},
  {"role": "user", "content": "Tell me about AI"}
]

```
**Role descriptions:**

- `user`: User message (use this most of the time)
- `system`: System prompt to set AI behavior and role
- `assistant`: AI‚Äôs previous responses, used for conversation context

[‚Äã](#param-temperature)temperaturenumberControls output randomness, range 0-2

- Lower values (e.g., 0.2) make output more deterministic
- Higher values (e.g., 1.8) make output more random

Default: 1.0
[‚Äã](#param-max-tokens)max\_tokensintegerMaximum number of tokens to generateDifferent models have different maximum limits, please refer to specific model documentation
[‚Äã](#param-stream)streambooleanWhether to use streaming output

- `true`: Streaming response (SSE format)
- `false`: Complete response at once

Default: false
[‚Äã](#param-top-p)top\_pnumberNucleus sampling parameter, range 0-1Controls diversity of generated text, recommend using either this or temperatureDefault: 1.0
[‚Äã](#param-frequency-penalty)frequency\_penaltynumberFrequency penalty, range -2.0 to 2.0Positive values reduce the likelihood of repeating the same wordsDefault: 0
[‚Äã](#param-presence-penalty)presence\_penaltynumberPresence penalty, range -2.0 to 2.0Positive values increase the likelihood of talking about new topicsDefault: 0
[‚Äã](#param-stop)stopstring or arrayStop sequencesUp to 4 sequences where generation will stop when encountered
[‚Äã](#param-n)nintegerNumber of completions to generateDefault: 1**‚öÝÔ∏è Note:** Must enter a plain number (e.g., `1`), do not use quotes or it will cause an error
