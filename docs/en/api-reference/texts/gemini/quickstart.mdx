[‚Äã](#param-model)modelstringrequiredModel nameThe examples use `gemini-2.5-pro`, which you can replace with other supported Gemini models:

- `gemini-2.5-flash` - Gemini 2.5 Flash
- `gemini-2.5-pro` - Gemini 2.5 Pro

- `gemini-2.5-pro-thinking` - Gemini 2.5 Pro Deep Thinking

[‚Äã](#param-method)method enum (string) required Generation method (recommended: `generateContent` for quick start):

- `generateContent`: Wait for complete response and return at once
- `streamGenerateContent`: Stream response, return content in chunks

Available options: `generateContent`, `streamGenerateContent`
