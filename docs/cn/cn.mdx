## [​](#sdk-与代码示例)SDK 与代码示例

官方 SDK 支持 Python、Node.js 和 Java。所有 SDK 均 OpenAI 兼容，仅需更改 Base URL 配置即可无缝迁移。
### [​](#python-sdk)Python SDK

通过 `pip install openai` 安装。设置 Base URL 并使用现有 OpenAI 代码，无需修改。
复制
```shiki
from openai import OpenAI
client = OpenAI(
    base_url="https://api.dashlyai.cc/v1",
    api_key="your-dashlyai-key"
)

```
### [​](#node-js-sdk)Node.js SDK

通过 `npm install openai` 安装。配置 baseURL 参数为 DashlyAI 端点，保留现有集成。
复制
```shiki
import OpenAI from 'openai';
const client = new OpenAI({
  baseURL: 'https://api.dashlyai.cc/v1',
  apiKey: 'your-dashlyai-key'
});

```
### [​](#java-sdk)Java SDK

OpenAI 兼容的 Java 客户端。简单配置更改即可访问所有 DashlyAI 模型和供应商。
## [​](#从-openai-迁移)从 OpenAI 迁移

5 分钟内切换到 DashlyAI。无需重写代码——只需更改 Base URL 和 API Key。
1

获取 API Key

注册 DashlyAI 并从[控制台](https://dashlyai.cc/console/token)获取 API Key。2

更新 Base URL

将 OpenAI Base URL 改为 DashlyAI 端点。现有 SDK 集成无需修改即可工作。3

测试端点

使用新的 Base URL 运行现有代码。所有 OpenAI 兼容端点响应格式完全相同。在控制台查看延迟和速率限制。

## [​](#常见问题)常见问题

如何从 OpenAI 迁移？

只需将 Base URL 更改为 `https://api.dashlyai.cc/v1` 并使用您的 DashlyAI API Key。保留现有 SDK——无需重写代码。迁移通常在 5 分钟内完成。

SLA 包含什么？

我们的企业 SLA 包括 99.9% 可用性保证、全球 CDN 加速、多供应商路由自动故障转移和实时状态监控。包含速率限制管理和 Webhook 支持。

定价如何与 OpenAI 比较？

DashlyAI 提供透明的按 Token 定价，比官方供应商更实惠。按需付费无需订阅，企业用量享受批量折扣，无隐藏费用。
